# INTRODUCTION-TO-DATABASE-FINAL-PROJECT
The Ramsy Medical Laboratories Record System is an all-in-one clinic operations tool
for small to mid-sized medical facilities. It handles the full patient journey — from
walking in the door to viewing their complete medical history — while keeping a full
audit trail of every action taken by staff.
The system connects directly to a MySQL database and provides a clean, role-aware
desktop GUI that any staff member can use without technical knowledge.
